# Previsor de Preços de Imóveis com IA 🏠

Este é o meu primeiro projeto de Machine Learning! Desenvolvi um modelo de **Regressão Linear** que aprende a relação entre o tamanho de uma casa e o seu preço de mercado.

## 🚀 O que este projeto faz?
- Utiliza a biblioteca `Scikit-Learn` para treinar um modelo.
- Recebe a área em m² e retorna uma estimativa de preço em Reais.
- Gera um gráfico visual para mostrar a linha de tendência aprendida pela IA.

## 🛠️ Tecnologias utilizadas
- **Python**
- **Pandas** (Manipulação de dados)
- **Matplotlib** (Gráficos)
- **Scikit-learn** (O "cérebro" da IA)

## 📈 Resultado
O modelo foi capaz de traçar uma linha de regressão baseada na equação:
$$y = wx + b$$
